# react-antd-mall

通过React构建构建电商后台管理系统，利用网上现成的RESTful API来对接

## 技术选型

1. 前后端完全分离，分层架构，模块化开发
2. React v16.13.1,，Ant Design4 组件库，React-Router, ES6, Less

## 完成功能

1. 登录和退出登录功能
2. 商品、品类、订单、用户、列表渲染展示，并可进行分页
3. 可使用id或关键字对商品进行搜索，可以直接对商品进行上架或下架处理
4. 添加商品可以选择图片上传，支持富文本编辑
